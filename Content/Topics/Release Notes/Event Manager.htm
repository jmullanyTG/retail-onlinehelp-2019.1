Event Manager

The following enhancements are now available in the Event Manager solution.

Slot Set Layout Header

The Space name has been added to the Slot Header. Also, hover text is now available if the slot is too small to show the entire header.

Item Display Format Restrictions

The available Item Display Formats for a specific template can now be restricted. A new dropdown in the Slot Set Template configuration UI has been added to indicate when a template is used in an Event a user can only see the appropriate Item Display Formats. By default, all Item Display Formats are available for all slot set templates.
