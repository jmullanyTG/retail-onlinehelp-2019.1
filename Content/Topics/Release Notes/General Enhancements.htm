Inventory Forecast

There are new controls within the Forecast viewer that allow you to select which series you do and do not want to see and also to change the color controls within them.

There are two new user rights. The new user rights are:

Solution User Right
Forecasts Allow Series Configuration
Item Manager Allow Stock Projection Series Configuration

Fill Column right-click menu option DMDEV-58620

A new right-click menu option labeled Fill Column has been added to data grids. The new option allows users to select a value in a grid and populate all fields in the column with that value, only those fields above the value, or only those below the value. The option is available in grids where there are not dependent fields that need to be cross-filled and it is only enabled where it can be used. The following rules always apply:

  • The grid must be editable and not be read-only.
  • The focused column must be editable and not be read-only.
  • The selected cell must be editable.
  • The row must be a valid data row.

Color configuration option in graphs DMDEV-59509

Within Inventory Manager, individual users can now configure the color palette within the graphs on the Stock Holding Projection tab and the Forecast Viewer tab. This new functionality allows users to select a color palette which is most visually appealing and accessible for them. Select the new user rights below to enable this functionality.

Tab New User Right
Stock Holding Projection Allow Stock Projection Series Configuration
Forecast Viewer Allow Series Configuration

New workflows associated with moving data to and from Allocations and Assortment Planning DMDEV-43791

The Allocations solution can now use the plans created in the Assortment Planning solution. When an assortment plan is finalized and approved it can now be executed in the Allocations solution where a new column lets you allocate based on the assortment plan.
