Promotion Manager

The following promotion-related enhancements are now available.

Validation logs DMDEV-57481

Advanced filtering has been added to the Validation Log to let users review and mark as reviewed (override) validation failure records allowing Promotions to continue to downstream systems. Additionally, the Validation log now returns large record sets. The new filters are:

  • Error Encountered
  • Error Overridden
  • Warning Found
  • Warning Overridden
  • Information Provided
  • Information Overridden

Users can select multiple filters as required.

Promotion strategy restrictions DMDEV-55681 DMDEV-55681 DMDEV-55682

Users can now be restricted to which promotion strategies they can use via a new button labeled Edit (Edit User Access column) on the Promotion Strategy tab on the Promotion Configuration screen. To access the Promotion Configuration screen, select Influence > Promotion > Configuration. Click Edit to display the Remove User Rights screen in which you add or remove user access to particular promotion strategies.

Users will only have visibility to those promotion strategies for which they have access. When creating a new promotion, the strategies that display in the Promotion Strategy dropdown list on the Create New Promotion screen are filtered based on the user’s access.
