The following system policies have been added.

Module: Assortment DMDEV-64485

System Policy Name:

Allow supplemental commitments

System Name:

Assortment.OrderManagerAddAssortmentPlanCommit.AllowSupplementalCommits

Description:

Allows for supplemental commitment orders for a period which already contains a confirmed commitment order. The possible settings are:

  • 0 = Do Not Allow.
  • 1 = Allow Supplemental Commits and Enforce Constraints on Supplemental Commits.
  • 2 = Allow Supplemental Commits and Do Not Enforce Constraints on Supplemental Commits.
