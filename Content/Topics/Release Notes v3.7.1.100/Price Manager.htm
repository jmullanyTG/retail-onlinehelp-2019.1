Price & Markdown

The following enhancements are now available in the Price Manager solution.

Reviewing Validation failuresDMDEV-58706dmdev-50129-5

A new Validation Severity column displays a Validation failure severity state as well as which failures have been reviewed for:

  • Price Actions
  • Site Clusters
  • Variants
  • Item Masters
  • Inventories

The following are the Validation Severity icons for failures not yet reviewed:

Validation Severity Icon Description
Information Provided
Warning Encountered
Error Found

The following are the Validation Severity icons for failures that have been reviewed:

Validation Severity Icon Description
Information Reviewed
Warning Reviewed
Error Reviewed

Reducing the number of reviews of Validation failures DMDEV-50121

Users have the ability to approve (override) Validation failures and, therefore, reducing the number of reviews needed throughout the lifecycle of the Price Action.

There is a check box labeled Allow Validation Override for each of the following:

  • Information
  • Warning
  • Error

Filtering product Validation failures based on where the Validation Log originated DMDEV-58717

You can now filter product Validation failures based on where the Validation Log originated so that you only see specific product validation failures to speed up the review process.

Level Filter
Plan All Price Action Validation failures, at the Inventory level, for ALL Site Clusters
Price Action All Site Cluster Validation failures, at the Inventory level, for ALL Site Clusters
Site Cluster All Variant's Item Master's Inventory Validation failures for the specific Site Cluster
Variant All associated Item Masters' Inventory Validation failures for the specific Site Cluster
Item Master All associated Inventory Validation failures for the specific Site Cluster
Inventory Validation failures for the specific Site Cluster

Validation logs DMDEV-57489

New advanced filtering has been added to the Validation Log to let users review and mark as reviewed validation failure records allowing Promotions to continue to downstream systems. Additionally, the Validation log now returns large record sets. The new filters are:

  • Error Encountered
  • Error Overridden
  • Warning Found
  • Warning Overridden
  • Information Provided
  • Information Overridden

Users can select multiple filters as required.

Cloning price actions DMDEV-56492

Price actions can now be cloned on the Price Manager Actions screen. Clicking the Clone button lets users quickly create a new price action based on an existing price action. Users require the access to create price actions. All Price Action Strategy values will remain intact while the system evaluates any change to Cost and Price based on the new Price Action Start Date.

Price Strategy User\Role Restrictions DMDEV-56907DMDEV-56906

Users can now be restricted to which price strategies they can use via a new button labeled Edit (User Access column) on the Price Strategies tab on the Price Manager Configuration screen.

To access the Price Manager Configuration screen, select Influence > Price and Markdown > Configuration. On the Price Strategies tab, click Edit to display the Remove User Rights screen in which you add or remove user access to particular price strategies.

Users will only have visibility to those price strategies for which they have access. When creating a new Price Action, the strategies that display in the Price Action Strategy dropdown list on the Create New Price Action screen are filtered based on the userâ€™s access.

Restrictions on special characters removed DMDEV-57805

The constraint preventing users from including special characters in Pricing Plan or Price Action names has been removed.

Automatic saving of Pricing Plans DMDEV-56477DMDEV-56476

To improve the user experience, the system now automatically saves a Pricing Plan when users add or remove a Site using Filters.

Ability to copy Variant Codes from the Product Grid Control DMDEV-57673

A new right-click option labeled Copy Column Cell(s) lets users copy Variant Codes from the Product Grid Control. This allows for the creation of a new Price Action for only the copied variants.
