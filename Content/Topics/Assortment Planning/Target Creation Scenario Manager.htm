Target Creation

Within Scenario Manager, users can access scenarios relevant to an intersection of the plan product and site hierarchies and a window of time. Users can create as many scenarios as they wish, but only one can be in active use within a plan for a given intersection of product, site, and time. This screen helps users quickly identify scenarios currently in use through scenario status and conditional formatting, and also provides details about selected scenarios without having to edit them.

Ultimately, scenarios can be used to create an excellent foundation for assortment plans through a process called Smart Start.For more information, see Smart Start .
