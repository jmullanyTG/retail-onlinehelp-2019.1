The following enhancements are now available in the Inventory Manager solution.

Enhancements to Stock Holding Projection Graph DMDEV-58532

The Stock Holding Projection Graph has been enhanced as follows:

The Projection and Demand graphs have been combined into a single visualization called Projection. The grid remains unchanged. Within the Projection graph, users can select and de-select individual series. The graph re-draws based on the selected series. The following new system policy governs which series are returned to the Stock Holding Projection Graph:

  • Stock Holding Projection Graph

Additionally,

  • The Calculate by Day button has been renamed Calculate.
  • The Calculate by Period button has been removed.
  • A new button labeled Advanced Calculate has been added to the Stock Holding Projection screen. Clicking the Advanced Calculate button opens up the Replenish Task dialog. The new Advanced Calculate button is controlled by the following new user right:
    • Allow Stock Projection Calculation Using Advanced Calc button

Forecast Fulfillment Split Factors view DMDEV-58519

A new tab labeled Forecast Fulfillment Split Factors has been added to the to the Item Manager solution. This tab shows the calculated fulfillment split factors. The split factors are displayed based on Item Master and are read-only. If fulfillment data exists, then checking the "Show Zero Values" parameter displays any destination sites that have that Item Master but don't have a split factor. If fulfillment data doesn't exist, then the pivot grid only shows the Product Description column while Site, Destination, and Split Factors are all null.
