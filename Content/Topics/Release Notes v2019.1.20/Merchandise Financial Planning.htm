Merchandise Financial Planning

The following enhancements are now available in the Financial Planning solution.

% contribution metrics for components

Users can now more efficiently plan sales and margin impacts by using new % contribution metrics for comp sales, non-comp sales, regular sales, promo sales, and markdown sales.

DMDEV-57500

The new metrics are:

  • Comp Sales Contribution %
  • Comp Sales Units Contribution %
  • Non-Comp Sales Contribution %
  • Non-Comp Sales Units Contribution %
  • Reg Sales Contribution %
  • Reg Sales Units Contribution %
  • Promo Sales Contribution %
  • Promo Sales Units Contribution %
  • Markdown Sales Contribution %
  • Markdown Sales Units Contribution %
