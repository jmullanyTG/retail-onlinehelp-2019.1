Changes

Version 3.7.1 included the following major changes:

  • Auto Lock on Cell Change option. Sales-related cell values can now lock themselves automatically after you update them. When the new Auto Lock on Cell Change is enabled, each time you change the value of a sales-related metric in the grid, the system automatically locks the changed cell as well as its related sales cells. This saves you from having to stop and manually click Lock after each change, which is especially convenient when you want to to change a number of values in a plan. For more information, see Auto Lock on Cell Change.
  • Restore Data Layout feature. A new Restore Data Layout option replaces the data fields displayed in the grid with those from another layout without changing the current pivots and filters. This lets you easily switch out metrics as you work without having to recreate your current product/location/time dimensions and filters. For more information, see Changing Data Layouts.
  • New folder structure for organizing layouts. You can now save layouts in groups, reducing the clutter of the Restore Layout menu and making it easy to always find the layout you want to use. For more information, see Managing Layout Groups.
  • Consolidation of planning views into new Hybrid Financial Planning Grid. A singe new planning view replaces the 3 planning views (Main Planning View, Comp/Non-Comp Planning View, and Sales By Type Planning View) that appeared in previous versions. For more information, see The Hybrid Financial Planning Grid.
  • DC stock can be split by supply chain. The logic related to how stock is split among DCs has been improved. Now, a plan's supply chain is used to determine how stock should be split among DCs. For more information, see DC Stock Splits.

Version 2019.1 included the following change:

% contribution metrics for components

Users can now more efficiently plan sales and margin impacts by using new % contribution metrics for comp sales, non-comp sales, regular sales, promo sales, and markdown sales. The new metrics are:

  • Comp Sales Contribution %
  • Comp Sales Units Contribution %
  • Comp Sales Units Contribution %
  • Non-Comp Sales Contribution %
  • Non-Comp Sales Units Contribution %
  • Reg Sales Contribution %
  • Reg Sales Units Contribution %
  • Promo Sales Contribution %
  • Promo Sales Units Contribution %
  • Markdown Sales Contribution %
  • Markdown Sales Units Contribution %
