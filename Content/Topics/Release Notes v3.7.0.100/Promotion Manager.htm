Promotion Manager

The following promotion-related enhancements are now available.

Identifying which Price Clusters with Validation failures have been reviewed DMDEV-50113

A new Validation Severity column on the Promotion List screen and in the Price Cluster grid displays a Promotion's Validation failure severity state as well as which failures have been reviewed.

The following are the Validation Severity icons for failures not yet reviewed:

Validation Severity Icon Description
Information Provided
Warning Encountered
Error Found

The following are the Validation Severity icons for failures that have been reviewed:

Validation Severity Icon Description
Information Reviewed
Warning Reviewed
Error Reviewed

Validation logs DMDEV-57481

Advanced filtering has been added to the Validation Log to let users review and mark as reviewed (override) validation failure records allowing Promotions to continue to downstream systems. Additionally, the Validation log now returns large record sets. The new filters are:

  • Error Encountered
  • Error Overridden
  • Warning Found
  • Warning Overridden
  • Information Provided
  • Information Overridden

Users can select multiple filters as required.

Exporting Promotions that have a Validation Severity type of Error DMDEV-50110

The system does not allow Promotions with a failed Validation Severity type of Error to be exported. Price Actions with Validation failures of severity type Error are exportable to downstream systems when all Validation failures of severity type Error have been overridden.

The following validation states can be exported:

Validation Severity ID Description
Validated
Information Provided
Warning Encountered
Information Reviewed
Warning Reviewed
Error Reviewed

The following validation states cannot be exported:

Validation Severity ID Description
Error Found

Promotion strategy restrictions DMDEV-55681 DMDEV-55681 DMDEV-55682

Users can now be restricted to which promotion strategies they can use via a new button labeled Edit (Edit User Access column) on the Promotion Strategy tab on the Promotion Configuration screen. To access the Promotion Configuration screen, select Influence > Promotion > Configuration. Click Edit to display the Remove User Rights screen in which you add or remove user access to particular promotion strategies.

Users will only have visibility to those promotion strategies for which they have access. When creating a new promotion, the strategies that display in the Promotion Strategy dropdown list on the Create New Promotion screen are filtered based on the user’s access.

Enhancement of the Promotion View Limit message DMDEV-56475

To enhance the clarity of the message that displays when when five Promotions are open and a user attempts to create a new Promotion, the Promotion View Limit message has been revised as follows:

“The Promotion, just created, cannot be viewed at this time as there are five Promotions already open for editing. The Promotion can still be found in the Promotion list for modification after at least one Promotion is closed.”
