Configuration Migration

The following enhancements are now available in the Configuration Migration solution.

Promotion Strategies DMDEV-57371 DMDEV-56908

For imported Promotion Strategies, users will only have visibility to those promotion strategies for which they have access. When exporting Promotion Strategies, System Administrators now have the ability to restrict users from specific Strategies.

Information Provided validation severity icon DMDEV-50268

The Validation Severity icon labeled Information Provided has changed color from green to blue to follow Demand Management Validation Severity State icon standards.

Promoting Validation Set configuration DMDEV-50077 DMDEV50076

New functionality allows System Administrators to promote Promotion Validation Set configuration including the ability to override Validation failures.
