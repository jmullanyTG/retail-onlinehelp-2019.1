Event Manager

The following enhancements are now available in the Event Manager solution.

Adjusting Event Slot Set templates to align with corporate financial strategies DMDEV-55484

Event Slot Set templates can now be adjusted to provide a specific amount of area on a Slot Set to drive corporate margin through Promotional Product placement.

Saving or canceling changes to the Slot Set when renumbering Slots DMDEV-55312DMDEV-50404

When entering or exiting the Renumber Slots feature you are now prompted whether to save any changes. This helps prevent accidental changes and means you avoid having to restart the numbering process from the beginning when you have a large number of Slots.
