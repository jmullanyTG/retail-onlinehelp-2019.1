Price & Markdown

The following enhancements are now available in the Price Manager solution.

Filtering out Pricing Plans based on associated Price Action Status DMDEV-56493

A column has been added to the Price Planning grid on the Pricing Home screen that filters out Pricing Plans based on associated Price Action Status. The possible statuses are:

  • Calculation Required
  • Saving
  • Calculating
  • Error Found
  • Warning Encountered
  • Information Provided
  • Error Overridden
  • Warning Overridden
  • Information Overridden
  • Validated

Cloning price actions DMDEV-56492

Price actions can now be cloned on the Price Manager Actions screen. Clicking the Clone button lets users quickly create a new price action based on an existing price action. Users require the access to create price actions. All Price Action Strategy values will remain intact while the system evaluates any change to Cost and Price based on the new Price Action Start Date.

Price Strategy User\Role Restrictions DMDEV-56906

Users can now be restricted to which price strategies they can use via a new button labeled Edit (User Access column) on the Price Strategies tab on the Price Manager Configuration screen.

To access the Price Manager Configuration screen, select Influence > Price and Markdown > Configuration. On the Price Strategies tab, click Edit to display the Remove User Rights screen in which you add or remove user access to particular price strategies.

Users will only have visibility to those price strategies for which they have access. When creating a new Price Action, the strategies that display in the Price Action Strategy dropdown list on the Create New Price Action screen are filtered based on the user’s access.

Validation log DMDEV-57088

New filters have been added to the Validation Log to let users review and mark as reviewed validation failure records allowing Promotions to continue to downstream systems. Additionally, the Validation Log no longer limits the number of records returned. The new filters are:

  • Error Encountered
  • Warning Found
  • Information Provided

Users can select multiple filters as required.

Restrictions on special characters removed DMDEV-57805

The constraint preventing users from including special characters in Pricing Plan or Price Action names has been removed.

Automatic saving of Pricing Plans DMDEV-56477DMDEV-56476

To improve the user experience, the system now automatically saves a Pricing Plan when users add or remove a Site using Filters.

Creating reporting metrics for specific Pricing Plans and Price ActionsDMDEV-50306

New functionality allows for the creation of reporting metrics for Specific Plan and Actions.

Ability to copy Variant Codes from the Product Grid Control DMDEV-57673

A new right-click option labeled Copy Column Cell(s) lets users copy Variant Codes from the Product Grid Control. This allows for the creation of a new Price Action for only the copied variants.
